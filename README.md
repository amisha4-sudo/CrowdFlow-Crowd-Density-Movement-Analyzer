# CrowdFlow-Crowd-Density-Movement-Analyzer
CrowdFlow is a computer vision-based system that analyzes video streams to estimate crowd density and detect overcrowding conditions.

## Features
- Real-time people detection  
- Crowd counting  
- Overcrowding alerts  

## Tech Stack
Python, OpenCV, YOLO  

## Run
streamlit run app.py

## How It Works
- Captures video frames  
- Detects people using YOLO  
- Counts individuals and triggers alerts  

## Future Improvements
- Heatmap visualization  
- Multi-camera tracking  
- Edge deployment
